Create `scripts/run_inference.py` implementing batched inference with the following specifications:

**CLI arguments (follow evaluate_stage1.py pattern):**
- `--checkpoint` (required): Path to trained model checkpoint directory
- `--test-csv` (required): Path to test data CSV file
- `--output-dir` (default: outputs/inference): Output directory for results
- `--batch-size` (default: 8): Inference batch size
- `--max-source-length` (default: 512): Max tokenization length for source
- `--max-target-length` (default: 512): Max generation length
- `--seed` (default: 42): Random seed for reproducibility
- `--dry-run` (flag): Write empty CSV without loading model/data

**Dry-run mode behavior:**
```python
if args.dry_run:
    output_dir.mkdir(parents=True, exist_ok=True)
    csv_path = output_dir / "inference_results.csv"
    with open(csv_path, 'w') as f:
        f.write("id,akkadian_source,english_translation\n")
        f.write("1,<dry-run>,<dry-run>\n")
    print(f"Dry-run: wrote {csv_path}")
    sys.exit(0)
```

**Full inference mode:**
1. Validate checkpoint exists (check for config.json or model.safetensors)
2. Validate test CSV exists and has 'transliteration' column
3. Load model and tokenizer: `AutoModelForSeq2SeqLM.from_pretrained(checkpoint)`, `AutoTokenizer.from_pretrained(checkpoint)`
4. Set seeds using `training_utils.set_seeds(args.seed)`
5. Read test CSV: `df = pd.read_csv(args.test_csv)`
6. Process in batches:
   - Tokenize batch of transliterations with padding and max_length
   - Generate translations: `model.generate(input_ids, max_length=max_target_length, num_beams=4)`
   - Decode outputs: `tokenizer.batch_decode(generated, skip_special_tokens=True)`
7. Write results to `{output_dir}/inference_results.csv`:
   - Columns: `id,akkadian_source,english_translation`
   - id: row index (1-based)
   - akkadian_source: original transliteration from test CSV
   - english_translation: generated translation

**Error handling:**
- Missing checkpoint → `print(f"Error: checkpoint not found: {checkpoint}", file=sys.stderr); sys.exit(1)`
- Missing test CSV → `print(f"Error: test-csv not found: {test_csv}", file=sys.stderr); sys.exit(1)`
