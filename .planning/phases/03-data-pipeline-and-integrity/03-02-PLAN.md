Create scripts/create_dataloader.py with:

1. **Imports** (AGENTS.md order):
   - Standard: import argparse, import sys
   - Third-party: import torch, from torch.utils.data import Dataset, DataLoader, from transformers import AutoTokenizer
   - Local: from scripts.data_loader import load_stage_data

2. **Constants**:
   - SEED = 42
   - DEFAULT_BATCH_SIZE = 4
   - DEFAULT_MAX_SOURCE_LENGTH = 256
   - DEFAULT_MAX_TARGET_LENGTH = 256

3. **Class TranslationDataset(Dataset)**:
   - __init__(self, df, tokenizer, max_source_length, max_target_length):
     - self.df = df
     - self.tokenizer = tokenizer
     - self.max_source_length = max_source_length
     - self.max_target_length = max_target_length
   - __len__(self): return len(self.df)
   - __getitem__(self, idx):
     - row = self.df.iloc[idx]
     - source = row['transliteration_normalized']
     - target = row['translation_normalized']
     - return {'source': source, 'target': target}

4. **Function collate_fn(batch, tokenizer, max_source_length, max_target_length)**:
   - sources = [item['source'] for item in batch]
   - targets = [item['target'] for item in batch]
   - inputs = tokenizer(sources, max_length=max_source_length, padding='max_length', truncation=True, return_tensors='pt')
   - with tokenizer.as_target_tokenizer():
     - labels = tokenizer(targets, max_length=max_target_length, padding='max_length', truncation=True, return_tensors='pt')
   - labels_tensor = labels['input_ids']
   - labels_tensor[labels_tensor == tokenizer.pad_token_id] = -100
   - return {'input_ids': inputs['input_ids'], 'attention_mask': inputs['attention_mask'], 'labels': labels_tensor}

5. **Function create_dataloader(stage: int, model_id: str, batch_size: int = DEFAULT_BATCH_SIZE, split: str = 'train', max_source_length: int = DEFAULT_MAX_SOURCE_LENGTH, max_target_length: int = DEFAULT_MAX_TARGET_LENGTH, seed: int = SEED) -> DataLoader:**
   - df = load_stage_data(stage, split, seed)
   - tokenizer = AutoTokenizer.from_pretrained(model_id)
   - dataset = TranslationDataset(df, tokenizer, max_source_length, max_target_length)
   - from functools import partial
   - collate = partial(collate_fn, tokenizer=tokenizer, max_source_length=max_source_length, max_target_length=max_target_length)
   - loader = DataLoader(dataset, batch_size=batch_size, collate_fn=collate, shuffle=False)  # shuffle=False for determinism
   - return loader

6. **Function main()**:
   - ArgumentParser with --stage (int, required, choices=[1,2,3]), --model-id (str, required), --batch-size (int, default=DEFAULT_BATCH_SIZE), --split (str, default='train'), --max-source-length (int, default=256), --max-target-length (int, default=256), --seed (int, default=SEED)
   - Parse args
   - loader = create_dataloader(args.stage, args.model_id, args.batch_size, args.split, args.max_source_length, args.max_target_length, args.seed)
   - Print summary: f"DataLoader created for stage {args.stage} ({args.split}): {len(loader)} batches of size {args.batch_size}"
   - Get first batch to validate: batch = next(iter(loader))
   - Print batch keys and shapes: print(f"Batch keys: {list(batch.keys())}, input_ids shape: {batch['input_ids'].shape}, labels shape: {batch['labels'].shape}")
   - Handle exceptions: catch all, print to stderr, sys.exit(1)

7. **Entry point**: if __name__ == '__main__': main()

Follow AGENTS.md: snake_case, type hints, 4-space indent, explicit error handling, single quotes.
